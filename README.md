# Secondhand Bot

A Telegram bot for posting and managing second-hand item listings, built with **aiogram 3**.

## Features
- Submit listings with photos/videos
- Admin vote-based approval
- Auto-publish to channel
- Ad posting support

## Setup
```bash
pip install -r requirements.txt
export BOT_TOKEN=your_bot_token
export ADMIN_ID=your_telegram_id
python bot.py
```
